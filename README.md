# 🎮 Number Guess Game (Flask)

A web-based number guessing game built using Python Flask.

## Features
- Random number generation
- Smart hints (Too small / Too large / Very close)
- 5 attempts limit
- Restart game option

## Tech Stack
- Python
- Flask
- HTML
- CSS

## Run Locally

Install Flask

pip install flask

Run the app

python app.py

Open browser

http://127.0.0.1:5000
